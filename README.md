## 项目简介

HbrayNG 面向 Android 设备，目标是把代理节点的获取、测试、整理和启动流程做得更直接。应用会在本地完成节点拉取、真实连通性检测和可用节点保留，方便用户快速获得可启动的 VPN 配置。

项目保持免费使用，不在 README 中公开内置节点来源。节点可用性会受到地区、运营商、网络环境和上游状态影响，应用内测试结果仅代表当次检测。

## 主要功能

| 功能 | 说明 |
| --- | --- |
| 一键获取测试 | 通过底部入口获取内置配置，并自动开始真实连通性测试 |
| 可用节点筛选 | 找到最多 5 个可用节点后停止测试，保留通过筛选的结果 |
| VPN 启动校验 | 测试阶段同步校验配置是否能生成可启动的 Xray VPN 配置 |
| 节点列表整理 | 每轮获取前清理旧 Hbray 节点，降低旧数据干扰 |
| 分应用代理 | 支持按应用选择代理或绕行，适合不同应用分流场景 |
| Telegram 推送 | 可将筛选后的可用节点通过 Telegram Bot 推送到指定聊天 |
| 中英文切换 | 应用内提供中英文界面切换入口 |

## 使用流程

1. 安装并打开 HbrayNG。
2. 首次启动 VPN 时，根据系统提示允许 VPN 权限。
3. 点击底部“获取测试”，等待应用获取并测试节点。
4. 测试完成后，应用会保留通过筛选的可用节点。
5. 回到节点列表，选择节点并点击启动按钮。
6. 如需分应用代理，可进入“分应用设置”选择要代理或绕行的应用。
7. 如需 Telegram 推送，可在菜单中填写 Bot Token 与 Chat ID 后执行推送。

## Telegram 推送

Telegram 推送会把当前保留的可用节点生成订阅文本，并通过 Telegram Bot API 发送出去。

| 配置项 | 说明 |
| --- | --- |
| Bot Token | 从 BotFather 创建机器人后获得 |
| Chat ID | 可填写个人、群组 ID，或频道用户名，例如 `@your_channel` |
| 权限要求 | 推送到群组或频道前，需要先添加 Bot，并允许它发送消息 |

Bot Token 只保存在手机本地设置中，不会写入仓库。公开推送前，请自行确认内容是否适合公开。

## 使用提醒

- 建议每次使用前重新执行“获取测试”，以获得当前网络环境下更可靠的节点结果。
- 如果 VPN 启动失败，先确认系统 VPN 权限已允许，再重新获取测试。
- 如果分应用代理未按预期生效，修改应用选择后建议重新启动 VPN 服务。
- 节点质量具有时效性，测试可用不代表长期稳定。
